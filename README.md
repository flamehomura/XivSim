# XivSim

## Engine

UE 5.7.4 Source Build

Branch:
release

Commit:
A Final Fantasy XIV Ultimate Raid Simulator featuring selected mechanics from The Omega Protocol (TOP), Futures Rewritten (FRU), and Dancing Mad (Ultimate). Designed and tested based on the Chinese mainland (CN) version of Final Fantasy XIV.

## Setup

1. Clone project
2. Open XivSim.uproject
3. Switch Unreal Engine Version
4. Generate Visual Studio Project Files
